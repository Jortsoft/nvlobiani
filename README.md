## 🎹 Keymaps

| Action                | Shortcut                              |
|-----------------------|----------------------------------------|
| 💾 Save file          | `<leader>space + S`                    |
| 📁 Toggle file tree   | `<leader>space + E`                    |
| 🔍 Find file          | `<leader>space + F`                    |
| 🔄 Switch files       | `<leader>space + P`                    |
| 📦 Import module      | `<leader>space + I`                    |
| ↩️ Undo               | `<leader>space + ←`                     |
| ↪️ Redo               | `<leader>space + →`                     |
